# Фронтенд социальной сети

Современный фронтенд для социальной сети, построенный на Vue.js 3.

## 🚀 Возможности

- 🔐 Регистрация и авторизация пользователей
- 📝 Создание, просмотр и удаление постов
- 👥 Главная страница с лентой всех постов
- 🛡️ Админ-панель для управления постами и пользователями
- 📱 Адаптивный дизайн

## 📋 Требования

- Node.js версии 16 или выше
- npm или yarn

## 🔧 Установка

1. **Установите зависимости:**

```bash
npm install
```

или с yarn:

```bash
yarn install
```

2. **Настройка бэкенда:**

Убедитесь, что ваш FastAPI бэкенд запущен на `http://localhost:8000`

Если ваш бэкенд использует другой адрес, измените его в файле `vite.config.js`:

```javascript
proxy: {
  '/api': {
    target: 'http://ваш-бэкенд-адрес:порт',
    // ...
  }
}
```

3. **Запустите проект в режиме разработки:**

```bash
npm run dev
```

или с yarn:

```bash
yarn dev
```

Приложение будет доступно по адресу: `http://localhost:5173`

## 🏗️ Сборка для продакшена

```bash
npm run build
```

или с yarn:

```bash
yarn build
```

Готовые файлы будут в папке `dist/`

## 🎯 API Endpoints

Приложение взаимодействует со следующими эндпоинтами бэкенда:

### Авторизация:
- `POST /log/token-login` - Вход в систему
- `GET /log/me` - Получение данных текущего пользователя

### Пользователи:
- `POST /user/create_user` - Регистрация
- `GET /user/get_all_users` - Список всех пользователей (только для админов)
- `DELETE /user/delete_user` - Удаление пользователя (только для админов)

### Посты:
- `GET /post/get_all_posts` - Получение всех постов
- `POST /post/create_post` - Создание поста
- `DELETE /post/delete-post` - Удаление поста

## 👤 Роли пользователей

### Обычный пользователь:
- Просмотр всех постов
- Создание своих постов
- Удаление только своих постов

### Администратор (isAdmin = true):
- Все возможности обычного пользователя
- Доступ к админ-панели
- Просмотр списка всех пользователей
- Удаление любых постов
- Удаление пользователей

## 🔒 Аутентификация

Приложение использует JWT токены для аутентификации:
- Токен сохраняется в localStorage
- Автоматически добавляется к каждому запросу через Axios interceptor
- При истечении токена пользователь перенаправляется на страницу входа

## 🎨 Стилизация

Приложение использует чистый CSS с переменными и современными возможностями:
- Flexbox и Grid для layouts
- CSS переменные для цветов
- Адаптивный дизайн с медиа-запросами
- Плавные анимации и переходы

## 🐛 Отладка

Если у вас возникли проблемы с CORS, убедитесь, что ваш FastAPI бэкенд настроен правильно:

```python
from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:5173"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

## 📝 Примечания

- Убедитесь, что бэкенд запущен перед запуском фронтенда
- При первом запуске создайте админа вручную через бэкенд (установите `isAdmin = True` в базе данных)
- Все пароли должны содержать минимум 6 символов
