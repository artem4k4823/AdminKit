# 🚀 Быстрая установка и запуск

## Шаг 1: Установка Node.js

Если у вас не установлен Node.js, скачайте и установите его с официального сайта:
https://nodejs.org/ (рекомендуется LTS версия)

Проверьте установку:
```bash
node --version
npm --version
```

## Шаг 2: Установка зависимостей проекта

Откройте терминал в папке проекта и выполните:

```bash
npm install
```

Это установит все необходимые пакеты (Vue.js, Vue Router, Pinia, Axios и т.д.)

## Шаг 3: Настройка подключения к бэкенду

**ВАЖНО:** Убедитесь, что ваш FastAPI бэкенд запущен на `http://localhost:8000`

Если ваш бэкенд использует другой порт, откройте файл `vite.config.js` и измените настройки proxy:

```javascript
proxy: {
  '/api': {
    target: 'http://localhost:ВАCШ_ПОРТ',  // Замените на ваш порт
    // ...
  }
}
```

## Шаг 4: Настройка CORS на бэкенде

Убедитесь, что в вашем FastAPI приложении настроен CORS. Добавьте следующий код в main.py (или где создается приложение):

```python
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware

app = FastAPI()

# Настройка CORS
app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:5173"],  # Адрес фронтенда
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

# ... остальной код вашего приложения
```

## Шаг 5: Запуск проекта

```bash
npm run dev
```

После запуска приложение будет доступно по адресу: **http://localhost:5173**

## 🎉 Готово!

Откройте браузер и перейдите по адресу `http://localhost:5173`

### Первые шаги:

1. **Зарегистрируйтесь** - создайте новый аккаунт
2. **Войдите** - используйте созданные учетные данные
3. **Создайте пост** - на главной странице заполните форму и создайте свой первый пост
4. **Для доступа к админ-панели** - в базе данных установите `isAdmin = True` для вашего пользователя

## 📝 Проблемы и решения

### Ошибка "Cannot find module"
```bash
rm -rf node_modules
npm install
```

### Ошибка подключения к API
- Проверьте, что бэкенд запущен
- Проверьте настройки CORS
- Проверьте порт в vite.config.js

### Ошибка 401 при запросах
- Убедитесь, что вы вошли в систему
- Проверьте, что токен сохраняется в localStorage (откройте DevTools → Application → Local Storage)

### Не видно админ-панели
- Установите `isAdmin = True` для вашего пользователя в базе данных
- Выйдите и войдите снова

## 📞 Помощь

Если что-то не работает:
1. Проверьте консоль браузера (F12)
2. Проверьте терминал с запущенным фронтендом
3. Проверьте терминал с запущенным бэкендом
4. Убедитесь, что все порты правильные
