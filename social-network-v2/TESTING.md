# 🔍 Тестирование WebSocket и Real-Time

## ✅ ШАГ 1: Проверка WebSocket подключения

### Откройте консоль (F12):

После входа в систему вы должны увидеть:
```
🔌 Подключение WebSocket для пользователя: 1
✅ WebSocket подключен успешно!
✅ Обработчик зарегистрирован: newPost
✅ Обработчик зарегистрирован: deletePost
```

**Если не видите** - WebSocket не подключился, проверьте бэкенд!

## ✅ ШАГ 2: Проверка создания поста

1. Заполните форму "Создать пост"
2. Нажмите "Опубликовать"

### В консоли должно быть:
```
📤 Событие отправлено: new_post {post: {...}}
```

### Если ошибка:
- Откройте Network (F12 → Network)
- Найдите POST /api/post/create_post
- Смотрите статус и ответ

## ✅ ШАГ 3: Ping/Pong (каждые 30 сек)

```
📡 Ping отправлен
📡 Pong получен
```

## 🐛 Частые проблемы:

### "Ошибка создания поста"
- Бэкенд не запущен
- CORS не настроен
- Поля пустые

### WebSocket не подключается
- Проверьте ws://localhost:8000/chat/ws/{user_id}
- Бэкенд должен быть запущен
- Проверьте консоль на ошибки
